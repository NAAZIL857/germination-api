================================================================================
DEMARRAGE RAPIDE
================================================================================

1. INSTALLER LES DEPENDANCES:
   pip install -r requirements.txt

2. TESTER LES MODULES:
   python quick_test.py

3. DEMARRER L'API:
   python api.py
   OU
   start_api.bat

4. ACCEDER A L'API:
   - API: http://localhost:8000
   - Documentation: http://localhost:8000/docs

5. TESTER L'API:
   python test_api.py
   OU
   python example_client.py

================================================================================
ENDPOINTS PRINCIPAUX
================================================================================

POST   /predict                  - Prediction + recommandations
POST   /recommendations          - Recommandations uniquement
GET    /conditions/{seed_type}   - Conditions optimales
POST   /sensor-data              - Ajouter des donnees
GET    /predictions              - Historique des predictions
GET    /stats/{seed_type}        - Statistiques
GET    /health                   - Etat de l'API

================================================================================
BASE DE DONNEES
================================================================================

Fichier: germination.db (cree automatiquement)

Tables:
- sensor_data   : Donnees des capteurs
- predictions   : Historique des predictions

================================================================================
EXEMPLE D'UTILISATION
================================================================================

Python:
-------
import requests

response = requests.post("http://localhost:8000/predict", json={
    "seed_type": "mais",
    "temperature": 25,
    "soil_humidity": 70,
    "light_level": 8,
    "pH_du_sol": 6.2
})

result = response.json()
print(f"Score: {result['predicted_score']}")

curl:
-----
curl -X POST "http://localhost:8000/predict" \
  -H "Content-Type: application/json" \
  -d "{\"seed_type\":\"mais\",\"temperature\":25,\"soil_humidity\":70,\"light_level\":8,\"pH_du_sol\":6.2}"
