================================================================================
                    GUIDE RAPIDE (3 ETAPES)
================================================================================

ETAPE 1: GitHub
  1. Creez un compte sur https://github.com
  2. Creez un nouveau repository "germination-api"
  3. Uploadez tous les fichiers du projet

ETAPE 2: Render
  1. Creez un compte sur https://render.com
  2. Connectez votre GitHub
  3. Cliquez "New +" > "Web Service"
  4. Selectionnez votre repository
  5. Configurez:
     - Build: pip install -r requirements_deploy.txt
     - Start: uvicorn api:app --host 0.0.0.0 --port $PORT

ETAPE 3: Tester
  1. Attendez le deploiement (2-3 minutes)
  2. Ouvrez: https://votre-api.onrender.com/docs
  3. Testez avec: python test_api_en_ligne.py

================================================================================
                    EXEMPLE D'UTILISATION
================================================================================

Une fois deployee, utilisez votre API comme ceci:

Python:
-------
import requests

API_URL = "https://votre-api.onrender.com"

response = requests.post(f"{API_URL}/predict", json={
    "seed_type": "tomate",
    "temperature": 25,
    "soil_humidity": 75,
    "light_level": 11,
    "pH_du_sol": 6.4
})

print(response.json())


JavaScript:
-----------
const API_URL = "https://votre-api.onrender.com";

fetch(`${API_URL}/predict`, {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    seed_type: 'tomate',
    temperature: 25,
    soil_humidity: 75,
    light_level: 11,
    pH_du_sol: 6.4
  })
})
.then(res => res.json())
.then(data => console.log(data));
